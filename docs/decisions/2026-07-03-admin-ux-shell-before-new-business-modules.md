# Admin UX Shell Yeni Business Modullerinden Once Yapilacak

Tarih: 2026-07-03

## Karar

Yeni teklif/siparis veya kargo modullerine gecmeden once admin panel deneyimi profesyonel shell yapisina alinacak.

## Gerekce

Mevcut sistem teknik olarak DB-backed calisiyor, fakat admin panel beklentisi:

- Sol menu
- Merkezi dashboard
- Urun/stok/teklif/siparis akis gorunurlugu
- Daha sik ve kullanisli UX

Bu beklenti karsilanmadan yeni moduller eklemek, sistemi daginik hissettirecek.

## Sonuc

Siradaki ana is Faz 2.5 olarak belirlendi:

- Admin shell
- Sidebar
- Operasyon dashboard
- Ortak admin layout

Sonra Faz 3.1 ve Faz 3.2 ile urun UX ve bayi/firma akislari guclendirilecek.
