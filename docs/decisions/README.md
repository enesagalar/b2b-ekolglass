# Karar Kayitlari

Bu klasor, proje boyunca alinan kalici kararlar icindir.

Karar kaydi su durumda eklenir:

- Mimari yon degisirse.
- UX navigasyon yapisi netlesirse.
- Veritabani veya auth stratejisi degisirse.
- Entegrasyon yaklasimi degisirse.
- Proje sahibinin verdigi operasyonel karar kod davranisini etkilerse.

Dosya adi formati:

```text
YYYY-MM-DD-kisa-karar-basligi.md
```
